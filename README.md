# অসমীয়াত পঢ়ক (Assamese Reader)

AI-powered Assamese text-to-speech. Select text on any webpage → hear it read aloud.  
By AAII & Xobdo. Runs locally.

---

## 📝 বৰ্ণনা (Description)

অসমীয়া ভাষাৰ বাবে AI-ৰ সহায়ত টেক্সট-টু-স্পীচ মডেল।  
যিকোনো ৱেবপেজত অসমীয়া লিখনি চিলেক্ট কৰক আৰু শুনক।

অসমীয়াত পঢ়ক (Assamese Reader) is a free, AI-powered text-to-speech extension for the Assamese language.

---

## 🔊 SELECT & LISTEN

Highlight any Assamese text on any webpage.  
A **"পঢ়ক"** button appears — click it to hear the text read aloud in natural Assamese.

---

## 📄 FULL PAGE READING

Right-click anywhere → **"সম্পূৰ্ণ পৃষ্ঠা পঢ়ক"** to read an entire article.

Works with:
- Wikipedia  
- News sites  
- Blogs  
- Any Assamese text content  

---

## ⚡ FEATURES

- Sentence-by-sentence streaming with visual highlighting  
- Smart text cleaning  
  - Removes HTML tags  
  - Strips references  
  - Converts numbers → Assamese words  
- Speed control (slower ↔ faster)  
- Works on any website with Assamese text  
- One-time model download, cached for instant loading  

---

## 🧠 TECHNOLOGY

- FastSpeech2 + Multi-Band MelGAN neural TTS  
- Trained on ~4 hours of Assamese speech data  
- Runs locally using **ONNX Runtime Web (WebAssembly)**
